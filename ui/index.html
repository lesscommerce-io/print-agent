Konfiguracja

Wklej token z panelu LessCommerce (Ustawienia sklepu → Drukarki etykiet → Dodaj drukarkę). Albo kliknij „Otwórz w Print Agent" w panelu, a token wpisze się sam.

Konfiguracja przechowywana w ~/.config/lesscommerce/print-agent.json. Token jest jak hasło — chronimy plik OS-owymi uprawnieniami.
